Html

  • HTML is the standard markup language for Web pages.
  • With HTML you can create your own Website.

    • Sample Program

    <!DOCTYPE html>
    <html>
    <head>
    <title>My First HTML Page</title>
    </head>
    <body>
    <h1>Hello World!</h1>
    <p>This is a Sample Paragraph</p>
    </body>
    </html>

    Output of the Program

    Hello World!

    This is a Sample Paragraph
